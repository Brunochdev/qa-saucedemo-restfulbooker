# Test Plan — SauceDemo

# Objetivo — Objective

## Português
Validar as principais funcionalidades da aplicação SauceDemo através de testes manuais e automatizados, garantindo o correto funcionamento dos fluxos críticos da plataforma.

## English
Validate the main functionalities of the SauceDemo application through manual and automated tests, ensuring the correct behavior of critical platform flows.
